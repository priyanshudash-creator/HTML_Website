# Dresssite — Women's Dress & Kurti Shop

A simple static HTML e-commerce website for browsing and shopping women's ethnic wear, kurtis, frocks, and dresses online.

----

## 📌 Project Overview

**Dresssite** is a front-end only shopping website built using plain HTML. It showcases a collection of women's dresses and kurtis with discount offers, 
product images, and individual product pages. The brand is powered by **Bluethread Brands Pvt Ltd** and manufactured by **Piyush Craft**, Bhubaneswar, India.
